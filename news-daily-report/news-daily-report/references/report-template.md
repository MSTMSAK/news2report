## 格式规范

1. **标题层级**：只用 `#` `##` `###` 三个层级，不要更深
2. **加粗**：仅用于小标题和关键术语，不要整段加粗
3. **热点概述**：每个热点的一句话概述控制在40字以内
4. **分析深度**：每个事件的"背景与动因"+"直接影响"+"深层含义"合计不少于150字
5. **趋势判断**：每个趋势方向至少引用2个具体事件/数据作为支撑
6. **风险/机会**：每个提示必须有明确的逻辑链，不能只是陈述现象
